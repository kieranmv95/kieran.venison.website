I have covered this in past articles on React and how I feel about the Context API and if it is replacing redux and if redux is dead so please [check that article out](/newsletter/2020/newsletter-2/)!

So in the last course section (14), I started building a small custom express API which can handle authentication using JSON web tokens. This is only in its most basic form at the moment and it's not yet hooked up to anything such as firebase.

The real meat of Authentication comes in section 15 which is 7 hours worth of content alone, so I'm sure I will be writing much more about that next week!
