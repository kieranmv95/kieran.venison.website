As I mentioned in the breakdown I have spent a lot of time this week trying to identifying what skill gaps I have and how I can go about filling these gaps.

To identify the gaps I wrote down a list of technologies. Technologies I know, technologies I have used a small amount and technologies I want to know.

Obviously this list is not extensive and its not got every skill I know in it such as CSS, HTML etc. This mainly contains all the things I want to focus on going forward.

The reason I split it up into the groups I have done above is so that I know where I need to put my focus. Things in the 'I want to know' column will need much more focus than the 'I have used' column as they have no foundations I can build upon.

The things in the 'I know' column are in there because they change a lot, so I have to keep updated and its essentially an ever evolving learning process.
